HyperText Markup Language (HTML) is the main markup language for creating web pages and others
information that can be displayed in web browser.

HTML is written in the form of HTML elements consisting of tags inclosed in angel brackets (like < html >). within the web page content. HTML tags most commonly come in pairs like < h1 > and , Although some tags represent empty elements and so are unpaired, for the example < img >. The first tag in a pair is start tag, and the second tag is the end tag (they are also called opening tags and closing tags). In between these tags web designers can add text, further tags , comments and other types of text based content.
